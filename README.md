# CaseDeep

案件管理 AI 對話系統 - 包含前後端的完整專案

## 專案結構

```
CaseDeep/
├── backend/    # Spring Boot 後端 API
├── frontend/   # React + Vite 前端應用
└── README.md
```

## 技術棧

### 後端 (Backend)
- Java 17+
- Spring Boot
- Spring Security + JWT
- JPA / Hibernate
- H2 / MySQL Database

### 前端 (Frontend)
- React 18
- Vite
- (更多功能開發中...)

## 快速開始

### 後端啟動
```bash
cd backend
./mvnw spring-boot:run
```

### 前端啟動
```bash
cd frontend
npm install
npm run dev
```
